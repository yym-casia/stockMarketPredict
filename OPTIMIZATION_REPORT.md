## 优化措施

### 1. 提高置信度阈值
- **原设置**: 无明确阈值
- **新设置**: 最小置信度 **0.85**
- **理由**: 0.85+置信度股票胜率66.7%，远高于整体42.9%

### 2. 调整止盈比例
- **原设置**: 8%
- **新设置**: **5%**
- **理由**: 实际平均最大涨幅仅3.03%，8%止盈难以达到，导致股票回撤后止损

### 3. 优化评分算法
- 增加成交量权重
- 置信度权重从固定改为动态计算
- 扩大筛选池从2倍到3倍，确保有足够高置信度股票

### 4. 更新配置文件
```yaml

  take_profit: 5.0    # 从8%降至5%
  stop_loss: 3.0      # 保持3%
  min_confidence: 0.85 # 新增
```

## 预期效果

1. **胜率提升**: 从42.9%提升至60%+
2. **止盈达成率**: 5%止盈更实际，预计止盈股票增加
3. **风险控制**: 低置信度股票被过滤，减少盲目交易
