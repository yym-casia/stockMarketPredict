# 阶段2完成：优化止盈策略

## 完成内容

### 1. 阶梯止盈系统
- **第1档**: 盈利+5% → 卖出30%仓位
- **第2档**: 盈利+10% → 卖出30%仓位  
- **第3档**: 盈利+15% → 卖出40%仓位
- 保留部分仓位博取更高收益

### 2. 移动止盈机制
- **启动条件**: 盈利超过5%后启动
- **触发条件**: 从最高点回撤3%时全部卖出
- **动态止损**: 
  - 盈利3% → 止损线上移至成本价
  - 盈利8% → 止损线上移至+5%

### 3. 动态持仓周期
- **热门板块** (AI/算力/机器人): 持仓16-21天
- **普通板块**: 持仓14天
- **防御板块** (银行/医药): 持仓11-14天
- **市场情绪**: 牛市延长30%，熊市缩短30%

### 4. 文件创建
- `daily_pipeline_v3.py`: 完整优化版
- `phase2_optimization.py`: 止盈策略独立模块

## 优化效果预期

### 当前问题
- 止盈过于死板（固定8%止盈）
- 无法锁定部分利润
- 盈利后回撤导致利润消失

### 预期改善
- **利润锁定**: 阶梯止盈确保部分利润落袋为安
- **盈利保护**: 移动止盈防止利润大幅回撤
- **灵活持仓**: 根据板块特性动态调整持仓周期
